# Multilingual Restful API

### 시작하기전에

- 다국어를 처리하는 API Web Server
- `Restful` API 
- `Tree`구조를 이용한 카테고리의 구성
- `CSV파일` 업로드제공
- 업로드 시 `자동화` 배포 (S3등등)
    - \+ 배포 RESTful API 제공

### 사용기술

- `Swagger UI`를 이용해서 Restful Docs 처리
- `Juint`을 이용한 TDD 적용 
    - Restful API에선 TDD를 어떻게 적용할까 ?
- `Flyway`를 이용한 SQL 형상관리
    - 별개 프로젝트로 구성
- `Cache`를 이용한 부하처리
